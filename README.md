# LearningFlow
LearningFlow is a comprehensive study guide application designed to help students break down their syllabus and track their progress.
**Core Features Planned:**
* Syllabus reduction and topic breakdown
* Daily checklist generation for study topics
* Weekly, monthly, and exam-wise progress tracking
* Academic result storage and visualization

**Tech Stack:**
* Database: ShaktiDB (PostgreSQL)
* Backend: Python (FastAPI/Flask)
* Frontend: React.js
